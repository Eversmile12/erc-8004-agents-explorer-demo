# ERC-8004 Scanner Demo

A demo app to learn how to build an ERC-8004 agent scanner from scratch.

## What is ERC-8004?

ERC-8004 is a standard for registering autonomous AI agents on the Ethereum blockchain. This scanner lets you browse and explore agents registered on the protocol.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

-   Next.js 15 (App Router)
-   TypeScript
-   Tailwind CSS
-   shadcn/ui
